# 📚 Chrome Extension Dashboard 技能書索引 (Skill Index)

本目錄包含專案的核心開發規範與技術手冊，旨在為開發者與 AI Agent 提供統一的實作標準。

| 排序 | Skill 名稱 | 核心內容與功能說明 |
| :--- | :--- | :--- |
| **01** | **[專案全局概覽](./01-project-overview/SKILL.md)** | **架構總覽**：目錄結構、Manifest V3 配置、`options.js` 核心模組分區、命名規範與改動檢查清單。 |
| **02** | **[UI 設計與主題](./02-ui-design-guidelines/SKILL.md)** | **界面規範**：Monokai 雙主題系統、CSS 變數、SPA 分頁架構、三欄式佈局、響應式斷點與卡片密度模式。 |
| **03** | **[圖標處理邏輯](./03-icon-handling/SKILL.md)** | **資產管理**：多層級圖標抓取降級策略 (Management API → Path Guessing → Emoji Fallback) 與載入錯誤處理。 |
| **04** | **[擴充狀態管理](./04-extension-state-management/SKILL.md)** | **行為控制**：擴充功能啟用/停用邏輯、Background 事件監聽、用戶操作追蹤與批量操作保護機制。 |
| **05** | **[資料儲存持久化](./05-data-storage/SKILL.md)** | **資料基準**：`chrome.storage.local` 鍵值定義、JSON 資料結構、快照機制與資料匯入/匯出相容規範。 |
| **06** | **[群組與拖放系統](./06-group-system/SKILL.md)** | **功能模組**：功能群組與設備群組雙重架構、自訂群組管理、雙重篩選邏輯與 HTML5 拖放排序實作。 |

---

> [!TIP]
> **如何使用這些技能書？**
> 1. **新規範查閱**：在進行任何代碼修改前，請先查閱對應的 Skill 以確保符合專案架構。
> 2. **AI 指令參考**：當指派任務給其他 AI 時，可提示其閱讀 `.agent/skills/` 目錄內容作為 Context。
> 3. **開發同步**：若新增了重大功能或修改了底層邏輯，請同步更新對應的 `SKILL.md`。
