# 圖標說明

由於無法自動生成圖標檔案，請按照以下步驟手動建立圖標：

## 建立方式

1. 使用任何圖像編輯器（如 Photoshop、GIMP、或線上工具）
2. 建立以下三個尺寸的圖標：
   - `icon16.png` - 16x16 像素
   - `icon48.png` - 48x48 像素  
   - `icon128.png` - 128x128 像素

## 建議設計

- **背景色**：深灰色 (#2c3e50) 
- **圖案**：白色的齒輪圖標 ⚙️ 或管理器圖標 🎯
- **風格**：簡約、專業、符合Monokai主題

## 快速解決方案

如果您不想自己製作圖標，可以：

1. 從網路下載免費的管理器圖標
2. 使用 favicon 產生器
3. 暫時使用現有圖標檔案（複製其他擴充功能的圖標）

圖標不影響功能運行，您可以稍後再更換。
